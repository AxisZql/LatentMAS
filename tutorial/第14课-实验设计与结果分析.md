## 14.3 评估指标

### 14.3.1 准确率（Accuracy）

```python
accuracy = correct / total
```

**计算方式**：
```
准确率 = 正确预测数 / 总样本数
```

**示例**：
```
正确数: 85
总样本: 100
准确率: 85 / 100 = 0.85 = 85%
```

### 14.3.2 Token 消耗

**测量方式**：统计生成的总 Token 数

**对比指标**：
| 方法 | Token 消耗 | 相对 baseline |
|------|-----------|---------------|
| baseline | 100 | 100% |
| text_mas | 250 | 250% |
| latent_mas | 130 | 130% |

### 14.3.3 推理速度

```python
time_per_sample = total_time / num_samples
```

**测量方式**：记录处理时间和样本数

**对比指标**：
| 方法 | 时间/样本 | 相对 baseline |
|------|----------|---------------|
| baseline | 5.0s | 1.0x |
| text_mas | 15.0s | 3.0x |
| latent_mas | 7.5s | 1.5x |

### 14.3.4 效率指标

```python
efficiency = accuracy / token_consumption
accuracy_per_token = accuracy / tokens
```

**综合效率**：准确率高且 Token 消耗低 = 高效

## 14.5 结果解析

### 14.5.1 JSON 输出格式

```json
{
  "method": "latent_mas",
  "model": "Qwen/Qwen3-4B",
  "split": "test",
  "seed": 42,
  "max_samples": 100,
  "accuracy": 0.85,
  "correct": 85,
  "total_time_sec": 1234.5678,
  "time_per_sample_sec": 12.3457
}
```

### 14.5.2 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `method` | str | 方法名称 |
| `model` | str | 模型名称 |
| `split` | str | 数据集划分 |
| `seed` | int | 随机种子 |
| `max_samples` | int | 样本数量 |
| `accuracy` | float | 准确率 (0-1) |
| `correct` | int | 正确数 |
| `total_time_sec` | float | 总耗时（秒）|
| `time_per_sample_sec` | float | 单样本耗时（秒）|

### 14.5.3 解析脚本

```python
import json

def parse_results(file_path):
    with open(file_path, 'r') as f:
        # 最后一行为 JSON
        lines = f.readlines()
        for line in reversed(lines):
            line = line.strip()
            if line.startswith('{'):
                return json.loads(line)
    return None

# 使用
results = parse_results('output.log')
print(f"Accuracy: {results['accuracy']:.2%}")
print(f"Speed: {results['time_per_sample_sec']:.2f}s/sample")
```

## 14.7 常见问题排查

### 14.7.1 显存不足（CUDA OOM）

**错误信息**：
```
RuntimeError: CUDA out of memory. Tried to allocate 2.00 GiB (GPU 0; 15.78 GiB total)
```

**解决方案**：
```bash
# 1. 减小 batch size
--generate_bs 5

# 2. 使用更小的模型
--model_name Qwen/Qwen3-4B

# 3. 减小 max_new_tokens
--max_new_tokens 1024

# 4. 使用 vLLM（显存效率更高）
--use_vllm
```

### 14.7.2 模型下载失败

**错误信息**：
```
OSError: Could not load model model_name
```

**解决方案**：
```bash
# 1. 检查网络连接
# 2. 设置 HuggingFace 镜像
export HF_ENDPOINT=https://hf-mirror.com

# 3. 手动下载模型
git lfs install
git clone https://huggingface.co/Qwen/Qwen3-4B
```

### 14.7.3 结果为 NaN

**错误信息**：
```
accuracy: nan
```

**可能原因**：
- `max_samples = -1` 但数据集为空
- 答案提取失败

**解决方案**：
```bash
# 1. 指定具体样本数
--max_samples 100

# 2. 检查数据集是否正确加载
```

### 14.7.4 准确率为 0

**可能原因**：
- 模型输出格式与答案提取不匹配
- 答案提取正则表达式问题

**排查步骤**：
1. 查看原始输出：`raw_prediction` 字段
2. 检查答案提取逻辑
3. 手动运行一个样本验证

## 14.9 本课小结

### 实验设计原则

| 原则 | 说明 |
|------|------|
| **单一变量** | 每次只改变一个因素 |
| **对照实验** | 设置基线对照组 |
| **重复验证** | 使用不同种子多次运行 |
| **全面记录** | 记录所有配置和结果 |

### 评估指标

| 指标 | 计算方式 | 用途 |
|------|----------|------|
| **准确率** | correct / total | 效果评估 |
| **Token 消耗** | 统计生成 Token | 效率评估 |
| **推理时间** | total_time / samples | 速度评估 |
| **效率** | accuracy / tokens | 综合评估 |

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| CUDA OOM | 减小 batch_size、使用 vLLM |
| 模型下载失败 | 设置 HF 镜像 |
| 结果 NaN | 检查数据集、种子 |
| 准确率 0 | 检查答案提取逻辑 |

## 3.1 回顾与引入

上节课我们了解了项目结构，知道 `data.py` 负责从 HuggingFace 加载各类基准数据集。本节课我们将深入学习数据加载的设计模式和实现细节。

### 数据加载的重要性

```
数据 → 数据加载 → 模型 → 输出
         ↑
      统一的接口
```

数据加载模块是连接原始数据和模型的桥梁，它需要：
1. **统一格式**：不同数据集有不同格式，需要转换为统一结构
2. **按需加载**：使用生成器（Generator）节省内存
3. **答案标准化**：确保答案格式一致，便于评估

## 3.2 数据格式约定

项目定义了一个**统一的数据格式**，所有加载函数返回相同结构的字典：

```python
{
    "question": str,   # 问题文本
    "solution": str,   # 解答过程（可选）
    "gold": str        # 标准答案（用于评估）
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `question` | str | 是 | 问题的完整文本 |
| `solution` | str | 否 | 解答过程，部分数据集有 |
| `gold` | str | 是 | 标准答案，用于计算准确率 |

## 3.4 生成器模式

项目使用**生成器（Generator）**而非列表来返回数据：

```python
def load_gsm8k(...) -> Iterable[Dict]:
    ds = load_dataset("gsm8k", "main", split=split, cache_dir=cache_dir)
    for item in ds:
        # 处理每条数据
        yield {...}
```

### 为什么使用生成器？

| 对比 | 列表 | 生成器 |
|------|------|--------|
| 内存占用 | 全部加载到内存 | 按需加载 |
| 适用场景 | 小数据集 | 大规模数据集 |
| 迭代次数 | 可多次迭代 | 只能迭代一次 |

### Iterable vs Iterator

```python
from typing import Iterable  # 可迭代对象（可多次迭代）
# vs
from typing import Iterator  # 迭代器（只能迭代一次）
```

项目使用 `Iterable[Dict]`，因为生成器只能迭代一次，但你可以转换为列表后多次使用：

```python
dataset_iter = load_gsm8k()        # 返回生成器
dataset_list = list(dataset_iter)   # 转换为列表，可多次使用
```

### 3.6.1 load_gsm8k() —— 数学应用题

```python
def load_gsm8k(split: str = "test", cache_dir: Optional[str] = None) -> Iterable[Dict]:
    ds = load_dataset("gsm8k", "main", split=split, cache_dir=cache_dir)
    for item in ds:
        question = item["question"].strip()
        solution = item["answer"]
        gold = normalize_answer(extract_gold(solution))
        yield {"question": question, "solution": solution, "gold": gold}
```

**数据集特点**：
- 来源：OpenAI GSM8K 基准
- 任务：小学数学应用题
- 特点：答案格式为 `#### 答案`（如 `#### 42`）

**示例**：
```

solution: "James starts with 5 marbles. He buys 3 more. 5 + 3 = 8. #### 8"

```

        labels = choices["label"]      # ["1", "2", "3", "4"]
        texts = choices["text"]         # 选项文本

        ordered_lines = [f"{lab}: {text}" for lab, text in zip(mapped_labels, texts)]

### 3.6.5 load_mbpp_plus() / load_humaneval_plus() —— 代码生成

def load_mbppplus(split: str = "test", subset: str = None, cache_dir: Optional[str] = None) -> Iterable[Dict]:
    ds = load_dataset("evalplus/mbppplus", subset, split=split, cache_dir=cache_dir)
    for item in ds:
        # 构建代码生成任务的 prompt
        question = f"""Please provide a self-contained Python script that solves the following problem in a markdown code block:
```python
YOUR_PYTHON_CODE
```:
{item["prompt"]}
Your answer will be tested on test cases like:
{item["test_list"][0]}
{item["test_list"][1]}
{item["test_list"][2]}
"""
        answer = str(item["test"])
        gold = answer

### 3.6.6 load_medqa() —— 医学问答

```python
def load_medqa(split=None, subset=None, cache_dir=None):
    # 加载本地 JSON 文件
    ds = load_dataset("json", data_files="./data/medqa.json", split='train')
    for item in ds:
        question = item["query"]
        raw_answer = str(item["answer"])

        # 将数字答案映射为字母选项
        choice_map = {"0": "A", "1": "B", "2": "C", "3": "D"}
        for idx, op in enumerate(item['options']):
            if raw_answer in op:
                answer = choice_map[str(idx)].lower()
                break

        gold = normalize_answer(answer)
        yield {"question": question, "solution": answer, "gold": gold}
```

**数据集特点**：
- 来源：MedQA 数据集（USMLE-style）
- 任务：医学多选题问答
- 特点：本地 JSON 文件格式，需要特殊处理

**原始 JSON 格式**：
```json
{
    "query": "患者症状描述...",
    "options": ["选项0文本", "选项1文本", "选项2文本", "选项3文本"],
    "answer": "0"  // 答案索引（0=A, 1=B, 2=C, 3=D）
}
```

def load_my_dataset(
    split: str = "test",
    cache_dir: Optional[str] = None

    ds = load_dataset("owner/my_dataset", split=split, cache_dir=cache_dir)

## 3.9 常见问题与解决方案

### Q1：数据集加载很慢怎么办？

**原因**：首次加载需要下载数据集到缓存目录。

**解决方案**：
- 确保网络连接正常
- 设置国内镜像加速：`pip install -i https://mirrors.aliyun.com/simple/ datasets`

### Q2：内存不足怎么办？

**原因**：数据集太大，全部加载到内存。

**解决方案**：
- 项目已使用生成器，内存占用已优化
- 减小 `batch_size` 参数
- 使用 `max_samples` 限制样本数

### Q3：自定义数据集如何加载？

**方案**：参考 `load_medqa()` 的写法，加载本地文件：

```python
ds = load_dataset("json", data_files="./path/to/your/data.json", split="train")
```

## 3.10 本课小结

### 核心要点

| 要点 | 内容 |
|------|------|
| **统一格式** | 所有数据集都转换为 `{question, solution, gold}` 格式 |
